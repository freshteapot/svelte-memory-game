# What is this?
- A port of the vue-memory-game to svelte.
- Game based on a json file, instead of hard coded.

# Status
- Not quite finished porting it, but good enough for it to work.

# Development
## Install
```sh
npm install
```
## Develop
```sh
make run-server
```

## Deploy site
```sh
make build-site
```
